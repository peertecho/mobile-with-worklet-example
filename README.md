# Mobile With Worklet Example
- An example of mobile app with a Worklet running as a backend service
- UI and Worklet can send messages to each other

## Usage
Minimal app
```shell
cd minimal
npm i
npm run dev:ios
npm run dev:android
```

Default app from `npx create-expo-app@latest`
```shell
cd default
npm i
npm run dev:ios
npm run dev:android
```
